# Oracle Database V$SQL_MONITOR
This repository is a practical, hands-on project focused on Oracle database performance monitoring using the v$SQL_MONITOR dynamic performance view. It demonstrates how to track and analyze SQL execution in real time, identify long-running or resource-intensive queries, and understand database workload behavior more effectively.  The project includes real-world SQL examples, monitoring queries, and explanations of key performance metrics such as execution time, buffer gets, disk reads, and wait events. It is designed to help developers, DBAs, and learners gain deeper insight into how Oracle executes and optimizes SQL statements under different workloads.  By using v$SQL_MONITOR, users can observe live SQL execution details, troubleshoot performance bottlenecks, and improve query tuning skills in Oracle environments.
